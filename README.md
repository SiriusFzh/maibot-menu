# MaiBot 菜单插件

发送 `/菜单` 即可生成一张精美的菜单图片，列出麦麦的所有可用功能和指令。

## 功能

- 完全手动配置，只显示你自己添加的功能和指令
- 支持在 WebUI 插件配置页面中添加/编辑/删除功能分组
- 支持在功能分组下添加子项（指令 + 描述）
- 以卡片式图片展示，风格统一美观
- 图片渲染失败时自动降级为纯文本

## 安装

在 MaiBot WebUI 插件市场搜索 `maibot.menu` 安装，或手动将本仓库克隆到 `plugins/` 目录。

## 使用

在任何群聊中发送：

```
/菜单
```

麦麦会生成一张图片，列出你配置的所有功能和指令。

## 配置

在 MaiBot WebUI 的插件配置页面中，找到「菜单」标签页下的「功能列表」，即可：

1. **添加功能**：点「添加项目」输入功能名称（如"每日分析"）
2. **添加指令**：在功能下的「指令列表」点「添加项目」，填写格式 `命令 : 描述`
3. **删除**：鼠标悬停在项目上点击删除按钮

配置示例：

```toml
[[menu.features]]
name = "每日分析"
commands = [
    "/summary : 查看今天的群聊总结",
    "/summary 昨天 : 查看昨天的群聊总结",
    "/mysummary : 查看自己今天的个人总结",
]
```
